# 燃烧的煤炭块

<ItemImage id="anvilcraftextrapower:burning_coal_block" scale="3"></ItemImage>

- 在<ItemLink id="anvilcraft:heat_collector" />的发电范围中，燃烧的煤炭块会产生与红热方块相同的电能。
- 在上方放置可加热的方块，可以使它被加热至红热。
- 燃烧的煤炭块会在燃烧8分钟后自动熄灭，化为<ItemLink id="anvilcraftextrapower:ashes_block" />。

# 燃烧的镁块

<ItemImage id="anvilcraftextrapower:burning_magnesium_block" scale="3"></ItemImage>


- 在<ItemLink id="anvilcraft:heat_collector" />的发电范围中，燃烧的镁块会产生与白炽方块相同的电能。
- 在上方放置可加热的方块，可以使它被加热至白炽。
- 燃烧的镁块会在燃烧8分钟后自动熄灭，化为<ItemLink id="anvilcraftextrapower:magnesium_oxide_block" />。
- 如果上方为<ItemLink id="anvilcraft:overheated_ember_metal_block" />则保持该方块的超温状态。
- <ItemLink id="anvilcraftextrapower:magnesium_oxide_block" />难以被挖动，但易被爆炸破坏。

- 由<ItemLink id="anvilcraftextrapower:magnesium_block" />点燃生成
