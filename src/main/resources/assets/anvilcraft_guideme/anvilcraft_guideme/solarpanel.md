# 太阳能板

<ItemImage id="anvilcraftextrapower:solar_panel" scale="3"></ItemImage>

- 太阳能板是一种可以在阳光下产生电能的发电设施。

# 机制

- 太阳能板在阳光下会产生电能，电力大小取决于阳光的强度。
- 太阳能板会通过视线检测(32格范围)来判断它是否能被太阳照射。
- 当太阳能板合上时，说明它不能被太阳照射，因此它不会产生电能。
