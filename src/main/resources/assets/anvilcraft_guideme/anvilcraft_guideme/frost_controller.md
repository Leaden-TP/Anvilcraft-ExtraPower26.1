# 浮霜控制器
<ItemImage id="anvilcraftextrapower:frost_controller" scale="3"></ItemImage>

浮霜控制器类似于控制棒，主要用于控制和削弱<ItemLink id="anvilcraftextrapower:uranium_rod" />

# 用途

- (1). 浮霜控制器可以用于控制<ItemLink id="anvilcraftextrapower:uranium_rod" />，可以代替<ItemLink id="anvilcraftextrapower:nuclear_collector" />控制<ItemLink id="anvilcraftextrapower:uranium_rod" />
-  在浮霜控制器3x3x3范围内即使没有<ItemLink id="anvilcraftextrapower:nuclear_collector" />也可以控制<ItemLink id="anvilcraftextrapower:uranium_rod" />不产生辐射
- (2). 单个浮霜控制器可以控制3x3x3范围内的<ItemLink id="anvilcraftextrapower:uranium_rod" />的发电值下降1，若4个浮霜控制器同时控制它，
-  则可彻底使 <ItemLink id="anvilcraftextrapower:uranium_rod" /> 完全失效停止发电
- (3). 浮霜控制器可以使5x5x5范围的水结冰，但当周围有<ItemLink id="anvilcraftextrapower:uranium_rod" />时禁用该功能
- (4). 浮霜控制器在接入红石信号时停用所有功能
