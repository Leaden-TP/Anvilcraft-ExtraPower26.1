# 铁砧投影机

<item id="anvilcraftextrapower:anvil_projector"/>

- 铁砧投影机适合进行大面积的铁砧加工。

# 合成

<recipe id="anvilcraftextrapower:anvil_projector"/>

# 机制

- 当被铁砧砸击时，产生下落高度-1的铁砧虚影。
- 铁砧虚影的样子由投影机后方的方块决定，如果方块是空气或不适合投影则产生普通幻灵铁砧投影。
- 如果虚影为特殊铁砧，则继承其特殊属性。
- 所有虚影铁砧的下落高度恒为1。
