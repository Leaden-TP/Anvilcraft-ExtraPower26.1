# 燃烧的煤炭块

<item id="anvilcraftextrapower:burning_coal_block"/>

- 在<translate key="item.anvilcraft.heat_collector"/>的发电范围中，燃烧的煤炭块会产生与红热方块相同的电能。
- 在上方放置可加热的方块，可以使它被加热至红热。
- 燃烧的煤炭块会在燃烧8分钟后自动熄灭，化为<translate key="block.anvilcraftextrapower.ashes_block"/>。

# 燃烧的镁块

<item id="anvilcraftextrapower:burning_magnesium_block"/>


- 在<translate key="item.anvilcraft.heat_collector"/>的发电范围中，燃烧的镁块会产生与白炽方块相同的电能。
- 在上方放置可加热的方块，可以使它被加热至白炽。
- 燃烧的镁块会在燃烧8分钟后自动熄灭，化为<ref item="anvilcraftextrapower.magnesium_oxide_block"/>。
- 如果上方为<ref item="anvilcraft.overheated_ember_metal_block"/>则保持该方块的超温状态。
- <ref item="anvilcraftextrapower.magnesium_oxide_block"/>难以被挖动，但易被爆炸破坏。
